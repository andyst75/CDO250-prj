# Скрипты для получения и обработки данных


### get_noaa_data.sh

Получает данные метеостанций за указанный год. Полученные данные помещаются в папку `../data/noaa`

Примеры использования:

    Получение данных за текущий год.
        bash ./get_noaa_data.sh

    Получение данных за 2012 год.
        bash ./get_noaa_data.sh 2012


### apply_noaa

Преобразование данных для дальнейшего использования (усреднение до областей)

Примеры использования:

    python3 apply_noaa.py --noaa-location ../data/noaa/gsod_2015.tar --output-csv ../data/temp/2015.csv


### get_genconsume.py

Обновление данных генерации/потребления

Примеры использования:

    python3 get_genconsume.py


### get_sber_consumer.py

Обновление данных индекса потребительской активности

Примеры использования:

    python3 get_sber_consumer.py
